Age and Wealth Distribution among Billionaires in Top 5 States in US

The Scatter plot shows the age versus the Networth of the Billionaires in the top 5 state with the most number of Billionaires. The Scatter plot if not effective in communicating the results as the data points are overlapped and densly distributed below 20 Billion. Selection Interval is used as interaction fixed to the Age variable in the scatter plot and adapted to the bar chart below. The user can now select the age interval and the number people distributed in those interval with respect to the state are shown in the bar graph. Interaction has increased the effectiveness of the plot.
